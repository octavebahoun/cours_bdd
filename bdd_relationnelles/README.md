# Sommaire du cours — Bases de données relationnelles

Ce dépôt regroupe des ressources pédagogiques pour comprendre et maîtriser les bases de données relationnelles. Chaque fichier aborde une étape clé du modèle relationnel, de la modélisation conceptuelle à la normalisation.

---

## Organisation des fichiers

| Fichier | Description |
|--------|-------------|
| [`modeledebdd.md`](./modele_de_bdd.md) | Introduction aux modèles de bases de données : concepts, entités, attributs, types de relations. |
| [`modele_relationnel.md`](./modele_relationnel.md) | Passage au modèle relationnel : clés primaires/étrangères, types de relations, contraintes d'intégrité. |
| [`normalisation.md`](./normalisation.md) | Principes et applications des principales formes de normalisation. |
| [`exercice.md`](./exercice.md) | Création et déclaration d'un schéma relationnel. |

---

## Objectifs pédagogiques

- Comprendre les fondements du modèle relationnel.
- Identifier les bonnes pratiques de modélisation.
- Appliquer les formes normales pour structurer efficacement les données.
- Manipuler les relations entre entités via SQL et diagrammes.
